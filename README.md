# 4192 El Camino Real — Mixed Use Development

Interactive floor plan viewer for a proposed mixed-use condo/apartment building in Palo Alto, CA.

**[View the plans →](https://meganho456.github.io/pa-design)**

## About the project

This is a design by SJA Development LLC / Design Division for a mixed-use building at 4192 El Camino Real, Palo Alto, CA. The project includes ground-floor amenities, two commercial units, and twelve residential apartments across four floors.

| Floor | Program |
|---|---|
| Ground / Site | Parking, lobby, fitness & wellness, storage |
| 2nd floor | 2 commercial units (1,550 sf + 1,533 sf) |
| 2nd–4th floors | 12 residential units (1-bed and 2-bed) |

Total residential units: **12** &nbsp;·&nbsp; Commercial units: **2** &nbsp;·&nbsp; Total parking: **25.75 spaces**

## Using the viewer

- **Switch floors** using the tabs across the top
- **Zoom** with scroll wheel or pinch on touch screens
- **Pan** by clicking and dragging
- **Reset** the view with the ↺ button
